Candidati

Ora, quando vengono aggiunti gli offerte di lavoro, puoi accettare le candidature degli utenti

Curriculum Vitae degli utenti

Un utente registrato, nella pagina dell'offerta di lavoro deve solo cliccare sul pulsante "Invia Candidato", un utente non registrato, deve solo inserire il proprio nome, cognome ed e-mail, allegare il proprio CV e inviare

  • Notifica e-mail all'utente e all'agenzia

  • Controllo dei candidati nell'account personale dell'agenzia

  • Ottenere tutti i candidati tramite connessione API

  • Elenco dei CV degli utenti sul sito tramite connessione api

  • Dettagli del CV dell'utente tramite connessione api
